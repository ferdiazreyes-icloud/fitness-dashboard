# Spec: Health Dashboard

## What it does

A personal health and fitness dashboard that consolidates data from three sources into interactive visualizations accessible from any device.

## Data Sources

1. **Apple Health** (primary) — XML export from iPhone (~5 GB). Contains 8 years of data: heart rate, VO2Max, HRV, sleep, steps, calories, body composition, workout sessions, GPS routes. Processed by `scripts/etl_apple_health.py` using SAX streaming (~65 seconds).

2. **Strong App** — Excel export (.xlsx) from the gym logging app. Contains set-level detail: exercise name, weight (lb), reps, RPE. Processed by `scripts/parse_strong.py`.

3. **Claude Coach** — Weekly training plan CSV generated by Claude. Contains prescribed exercises, target weight/reps/RPE per set. Enables plan-vs-reality comparison.

## Dashboard Pages (5 current)

1. **Resumen Semanal** — KPIs (sessions, km, minutes, kcal) with week-over-week comparison. Session table, weekly trend charts, activity type distribution.

2. **Running Analytics** — Pace evolution, running power scatter, distance distribution, monthly km, recent sessions table.

3. **Tendencias de Salud** — VO2Max trend, resting HR with 7-day rolling average, HRV (SDNN), sleep breakdown (deep/REM/core), SpO2.

4. **Composición Corporal** — Weight, BMI, body fat %, lean mass trends over time.

5. **Métricas Acumuladas** — Lifetime and YTD totals, monthly hours by activity type (stacked bar), weekly pattern, average sleep.

## User Stories

- As Fernando, I want to see my training volume this week compared to last week, so I can gauge consistency.
- As Fernando, I want to track my VO2Max and HRV trends, so I can monitor cardiovascular fitness.
- As Fernando, I want to see my running pace evolution, so I can track running performance over time.
- As Fernando, I want to view body composition trends, so I can monitor progress toward fitness goals.

## Pending Features

- **Adherence page** — Compare training plan vs actual (Strong log), with visual traffic light.
- **Alerts** — Notify if HRV drops >20% or no workout in 3+ days.
- **Exercise matching** — Normalize exercise names between plan and Strong.
- **Historical plans** — Accumulate weekly plans for progression tracking.
